# Construct Font Fixer
 
A Chrome extension that restores clean fonts on [Construct](https://construct.hackclub.com).
 
A recent commit swapped the site's default fonts to Papyrus and Comic Sans MS. This extension reverts that — silently, in the background.
 
---
# Install
 
1. Download the latest zip from [Releases](../../releases) and unzip it, or clone this repo
2. Go to `chrome://extensions`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** → select the `Construct Font Fixer` folder
 
Click the extension icon any time to toggle the fix on or off.
